== Description ==
After Activation the plugin , there will be a fixed green contact icon in left side of the theme and plugin also supports shortcode .
افزونه فارسی ارسال فرم تماس
Features:

* After Activation , there will be a fixed green contact icon in left side of the theme .
* Also use  [wp_simple_ajax_contact_form]  shortcode to show ajax contact form in the pages and posts.

Language Support:

* English
* Persian (فارسی)


== Installation ==
1. Upload `wp simple-ajax-contact-form` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Also you can use shortcode [wp_simple_ajax_contact_form]
